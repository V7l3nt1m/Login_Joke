# Login Joke é uma tela de Login com laravel, jquery e requisições ajax.
# O botão de login foge do usuário toda vez que os valores dos inputs não forem de acordo ao que está no banco de dados
# Punindo assim usuários teimosos.

Instalação
composer install
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan serve

No arquivo database/seeders/databaseeder.php constam os valores no banco de dados.

Divirta-se!!!
